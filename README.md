# Bolinho API (.NET)

API REST desenvolvida com **C# e ASP.NET Core** para gerenciamento de bolinhos.

---

## Sobre o projeto

Este projeto foi criado com o objetivo de praticar conceitos de desenvolvimento backend utilizando:

* C#
* ASP.NET Core Web API
* Estrutura de Controllers e Models
* CRUD completo (Create, Read, Update, Delete)

---

## Funcionalidades

* Criar registros
* Listar registros
* Atualizar registros
* Deletar registros

---

## Tecnologias utilizadas

* .NET 8
* ASP.NET Core
* C#
* Swagger (OpenAPI)

---

## Estrutura do projeto

```
Controllers/
Models/
Properties/
Program.cs
appsettings.json
```

## Como executar o projeto

### 1. Clonar o repositório

```
git clone https://github.com/Rafael-FullDev/bolinho-api-dotnet.git
```

---

### 2. Acessar a pasta

```
cd bolinho-api-dotnet
```

---

### 3. Rodar o projeto

```
dotnet run
```

---

### 4. Acessar a documentação (Swagger)

```
https://localhost:xxxx/swagger
```

---

## Endpoints principais

| Método | Rota        | Descrição   |
| ------ | ----------- | ----------- |
| GET    | /tasks      | Lista todos |
| POST   | /tasks      | Cria novo   |
| PUT    | /tasks/{id} | Atualiza    |
| DELETE | /tasks/{id} | Remove      |
